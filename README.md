# OfflineIRL-Music
This is the official implementation of Offline Inverse Reinforcement Learning for Structured Symbolic Music Generation.

The music demo page: [Demo](https://xmaple555.github.io/OfflineIRL-Music-Demo/)
# Environment 
* python:3.8.10
* python dependencies: `pip install -r requirements.txt`
